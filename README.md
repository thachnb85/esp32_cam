# Flashing Options:
- Board: ESP-32 Wrover Module
- Upload Speed: 115200
- Flash Frequency: 40 Mhz
- Flash Mode: QIO
- Partition Scheme: Huge App (3MB No OTA/1MB SPIFFS)
